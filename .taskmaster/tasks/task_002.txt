1. Firebase Authentication을 사용하여 인증 서비스를 생성합니다.
2. 이메일/비밀번호로 사용자 등록을 구현합니다.
3. 로그인 기능을 구현합니다.
4. 인증된 사용자를 위한 보호된 경로(protected routes)를 생성합니다.
5. 기본 정보가 포함된 사용자 프로필 페이지를 추가합니다.
6. 로그아웃 기능을 구현합니다.
7. 비밀번호 재설정 기능을 추가합니다.
8. Firestore에 추가 사용자 데이터(현재 학습 사이클, 날짜, 언어 환경설정)를 저장합니다.

Code structure:
```typescript
// auth.service.ts
export const registerUser = async (email: string, password: string) => {
  try {
    const userCredential = await createUserWithEmailAndPassword(auth, email, password);
    await setDoc(doc(db, 'users', userCredential.user.uid), {
      email,
      currentCycle: 1,
      currentDay: 1,
      preferredLanguagePair: 'korean-english',
      createdAt: new Date()
    });
    return userCredential.user;
  } catch (error) {
    throw error;
  }
};
```

# Test Strategy:
사용자 등록, 로그인, 로그아웃 흐름을 테스트합니다. 보호된 경로가 인증되지 않은 사용자를 리디렉션하는지 확인합니다. 비밀번호 재설정 기능을 테스트합니다. 사용자 데이터가 Firestore에 올바르게 저장되는지 확인합니다.
